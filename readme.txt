== Description ==
Techslidershow is the most powerful and user-friendly WordPress plugin for creating sliders.

It's fully responsive, and compatible with any WordPress theme. Create stunning sliders and add wonderful images and videos without writing a single line of code.

Enhance your blog or website with and Techslidershow, WordPress's most easy plugins. Simply drag and drop images, photos, videos, and other media from your WordPress Media Library into place, then customise the slide captions, links, and SEO sections all from one page.

Techslidershow is a WordPress image, photo, and video plugin that allows you to create a beautiful slider, slideshow, carousel, or gallery using the most basic and intuitive plugin interface of any custom image, photo, and video plugin.

The user can show off some latest work, photos, and videos, or even products from online store. People will have no trouble discovering your site and the slider, slide show, gallery, or carousel you make.

#### Feature List:
*Drag and Drop – It is an easy drag and drop of images that you have pre-made you can just add and arrange it as per your needs.
*Easy to use – The plugin is easy to use in any website or blog page.
*Call to action button – One an add any button on top of the images that can be either link to different page or website or a call-to-action button.
*Multiple Image and Video uploads – The user can add multiple images and videos as required and can change the look of the slider as well.
*Enable or disable – A certain images can be disabled for viewing if required for any custom page.
*Responsive – It supports all the devices and is responsive so that the user do not have to go for multiple design for multiple devices

#### Detailed Description: 

Techslidershow is an ultimate responsive image slider in WordPress, where the user can easily add number of multiple images, videos and can customize the order of display as well.

With this multiple image uploader, you can create several image slides in a single slider. It generates a wonderful ambience on your website with the slider image gallery it creates.

== Installation ==

### Automatic installation

1. Search for webslider through 'Plugins > Add New' interface.
2. Find the plugin box of webslider and click on the 'Install Now' button.
3. Then activate the webslider plugin.

### Manual installation

1. Download [webslider](https://downloads.wordpress.org/plugin/web-slider.zip)
2. Upload the webslider through 'Plugins > Add New > Upload' interface or upload webslider folder to the '/wp-content/plugins/' directory.
3. Activate the webslider plugin through the 'Plugins' menu in WordPress.
4. Create a new slider and add shortcode in page template.
